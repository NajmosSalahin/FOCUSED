## FOCUS — Pomodoro & Productivity Dashboard


### A fully featured productivity operating system built with pure frontend technologies.

FOCUS combines:

* Pomodoro timer systems
* Task tracking
* Habit management
* Daily analytics
* Productivity heatmaps
* Timeline visualization
* Goal tracking
* Project organization
* Weather + clock widgets
* Keyboard-first workflows

into a single high-performance dashboard.

Designed with a heavily customized Gruvbox-inspired UI and optimized for local-first usage.

# Installation

## Option 1 — Direct Download

Download:

```text
index.html
style.css
script.js
```

Place all three files in the same directory and open `index.html` in your browser.

---

## Option 2 — Git Clone

```bash
git clone https://github.com/yourusername/focus-dashboard.git
cd focus-dashboard
```

Then open:

```text
index.html
```

in your browser.
